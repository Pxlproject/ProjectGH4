CODE CONVENTIONS
----------------

- variabelen, bestandsnamen, ... : Engels
- commentaar: Nederlands
- 1 klasse/interface per bestand
- variabelen declareren: 
	--> public bovenaan
	--> dan protected
	--> vervolgens package (default)
	--> als laatste private
- 1 declaratie per lijn, zodat er nog plaats is voor eventuele comments aan het eind van de lijn
- witte lijn laten tussen methodes
- methoden beginnend met een hoofdletter, gevolgd door kleine letters, elk nieuw woord met een hoofdletter, gevolgd door haakjes () met of zonder parameters (vb. GetAuteurs(); )
- variabelen beginnend met een kleine letter, gevolgd door kleine letters, elk nieuw woord met een hoofdletter (vb. int i, double prijsPerEenheid, string naam)
- constanten bestaan uit hoofdletters, woorden gescheiden door '_' (vb. static final int MAX_PRIJS = 20)
- user controls afkorting control + naam, beginnend met kleine letters, elk nieuw woord beginnend met hoofdletter , vb. lblName, txtAge, btnOk

- if else statements als volgt:
	if (condition) 
	{
		statements;
	}

	
	if (condition) 
	{
		statements;
	} 
	else 
	{
		statements;
	}

	
	if (condition) 
	{
		statements;
	} 
	else if (condition) 
	{
		statements;
	} 
	else 
	{
		statements;
	}

- for statements als volgt:
	for (initialization; condition; update) 
	{
		statements;
	}

- while statements als volgt:
	while (condition) 
	{
		statements;
	}
	
- do while statements als volgt:
	do 
	{
		statements;
	} 
	while (condition);
	
- switch statements als volgt:
	switch (condition) 
	{
		case ABC:
			statements;
			/* falls through */
		case DEF:
			statements;
			break;
		case XYZ:
			statements;
			break;
		default:
			statements;
			break;
	}
	
	als er geen break; gezet wordt, voeg hier dan in de plaats een comment toe /* falls through */ om aan te duiden waar normaal een break moet staan
	
- try catch (finally) statements als volgt:
	try 
	{
		statements;
	} 
	catch (ExceptionClass e) 
	{
		statements;
	} 
	finally 
	{
		statements;
	}
